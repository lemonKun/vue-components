# vue-components
vue一些组件的封装与练习

## 安装：

基于vue-cli3快速原型开发,需要全局安装vue-cli3和vue-service-global

```bash
npm install -g @vue/cli@3
# OR
yarn global add @vue/cli@3
```

```bash
npm install -g @vue/cli-service-global@3
```

## 启动：

```bash
vue serve
```
